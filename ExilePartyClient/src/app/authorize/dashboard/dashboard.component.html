Getting started

To start using the app, enter a group-name of your choice in the top-left corner and press 'Enter'. Then tell your friends to enter the same group-name as you, and you will be able to see each other. Note that each player updates every time they enter a new zone.

Alpha information

This version is an alpha release, which means it's still in a very early stage. Bug reports and feature-requests are greatly appreciated as it will help us develop the app further, and are preferrebly posted at https://github.com/viktorgullmark/exile-party/issues. Updates will be applied automatically when you restart the program. Please be patient as it will download the new version and apply it before you can start using the application. To see what's next, make sure to check out upcoming changes in the readme file here.

Feel free to contact us on github or directly in-game for any questions/input.
